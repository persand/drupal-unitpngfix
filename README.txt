Overview:
--------
The Unit PNG Fix module enables Unit PNG Fix which adds PNG transparency support for IE6.
Read more about Unit PNG Fix in README-unitpngfix.txt or at http://labs.unitinteractive.com/unitpngfix.php.


Installation:
------------
1. Place this module directory in your modules folder (this will
   usually be "sites/all/modules/").
2. Go to "administer" -> "modules" and enable the module.
